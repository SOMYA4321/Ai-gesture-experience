# ✨ SOMYA KHANDELWAL – Gesture Experience

A futuristic **AI-powered gesture interface** built using **MediaPipe + JavaScript + Canvas**.  
This project allows users to interact with visuals using **hand gestures in real-time**.

---

## 🚀 Features

- 🎥 Real-time hand tracking using MediaPipe
- ✨ Interactive particle & visual effects
- ⚡ Gesture-based controls
- ✍️ Air writing (draw in air using finger)
- 💖 Shape detection (Heart, Peace, Star)
- 🌊 Shake to clear canvas
- 🎮 Multiple modes:
  - Play Mode
  - Shape Mode
  - Write Mode

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Advanced UI + Animations)
- JavaScript (Vanilla)
- MediaPipe Hands API
- Canvas API

---

## 🎮 How to Use

1. Open the website
2. Allow camera access
3. Use gestures:

### ✋ Play Mode
- Pinch → Shockwave
- Two hands → Lightning

### ❤️ Shape Mode
- Hands close → Heart
- Peace sign → ✌
- Fist → Star

### ✍️ Write Mode
- Point index finger → Draw
- Pinch → Stop drawing
- Shake hand → Clear canvas

---
